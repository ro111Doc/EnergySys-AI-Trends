# 图数据模型文档 data_model
## 一、文档说明
本文件基于课程Lesson5要求设计，严格贴合项目已导出的知网46条文献数据（含中文期刊、硕士论文、外文期刊），与`field_dictionary.md`字段口径保持一致，明确文献计量分析的核心实体（节点）、实体关联（边）及属性规范，为后续关键词共现、作者合作网络、机构影响力等分析提供统一数据标准，确保所有分析可复现、口径统一。

| 边类型         | 连接节点（起点→终点）       | 权重定义                     | 示例                                   | 方向性 | 说明                                                                 |

## 四、数据适配与约束说明
### 1. 数据适配性（基于现有文献数据）
- 已支持的节点/边：论文、作者、机构、关键词四类节点，及合作关系、共现关系等五类边，完全覆盖现有稳定导出字段（SrcDatabase、Title、Author、Organ、PubTime等）；
- 暂不支持的关联：因未导出References（参考文献）字段，暂不设计“引用关系（论文→论文）”，后续若补充该字段可按相同规范新增；
- 关键词补充方案：当前无Keywords字段，通过解析论文标题提取核心术语（如LSTM、BiLSTM、负荷预测、微电网），确保共现网络分析可行，弥补字段缺失影响。

### 2. 约束规则
- 唯一性约束：论文ID、作者ID、机构ID、关键词ID全局唯一，避免重复统计导致的分析偏差；
- 格式约束：发表年份统一提取为4位数值（如2024-06-30→2024），机构名称按`field_dictionary.md`要求标准化，作者姓名保留原始格式（中文姓名直接使用，外文姓名按“名+姓”统一）；
- 缺失值处理：DOI缺失的论文不影响节点创建，仅在关联分析时标注“无DOI”；Fund字段缺失不纳入节点属性（非核心分析维度），不影响核心计量指标计算。

## 五、适配分析场景
| 分析场景               | 依赖的节点-边组合                     | 核心用途                     |
|------------------------|---------------------------------------|------------------------------|
| 关键词共现热点分析     | 关键词节点 + 共现关系边               | 挖掘LSTM相关研究核心主题与前沿方向 |
| 作者合作网络分析       | 作者节点 + 合作关系边                 | 识别领域核心研究团队与合作模式     |
| 机构影响力分析         | 机构节点 + 所属关系边 + 发表关系边     | 统计核心研究机构发文量与行业分布   |
| 年度发文趋势分析       | 论文节点 + 来源关系边 + 发表年份属性   | 刻画2024-2025年研究活跃度变化     |
| 文献类型分布分析       | 论文节点 + 来源关系边                 | 对比期刊/硕士/外文期刊占比与质量   |
