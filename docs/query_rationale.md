# 检索式设计说明 query_rationale
## 1. 研究主题
本项目研究主题：**深度学习（LSTM）在电力负荷预测中的研究热点与趋势（2015–2025）**
采用文献计量方法，分析该领域的发文情况、研究热点、合作网络与发展趋势。

## 2. 检索式结构设计
本次检索采用标准四元组结构：
**方法词 AND 任务词 AND 场景词 NOT 排除词**
确保检索结果精准、不漏检、不误检。

## 3. 方法词（Method Terms）选择理由
方法词聚焦LSTM及其变体，覆盖领域主流方法：
- LSTM、长短期记忆网络、长短时记忆网络
- BiLSTM / 双向长短期记忆网络
- Attention LSTM / 注意力机制LSTM
目的：完整覆盖以LSTM为核心的深度学习负荷预测研究。

## 4. 任务词（Task Terms）选择理由
任务词围绕“电力负荷预测”展开，包含中英文常用表达：
- 电力负荷预测、负荷预测
- 短期负荷预测、STLF
- 电力需求预测、electric load forecasting、power load forecasting
目的：保证不同表达方式的相关文献都能被检索到，提高查全率。

## 5. 场景词（Context Terms）选择理由
场景词限定在电力领域：
- 电力系统、智能电网、电网、配电网
- power system、smart grid、power grid、distribution network
目的：确保研究场景限定在电力系统，不扩散到其他领域。

## 6. 排除词（Exclusion Terms）选择理由
排除明显不属于电力负荷预测的领域：
- 交通流预测、网络流量预测
- 服务器/CPU负载预测、桥梁荷载预测
目的：避免检索到结构相似但领域完全无关的文献，提升数据准确性。

## 7. 限定条件（Constraints）说明
- 时间范围：2015–2025（近10年，符合热点与趋势分析）
- 语言：中英文都保留（兼顾国内外研究）
- 文献类型：Article、Review、Thesis（保证文献质量）
- 检索字段：标题、摘要、关键词（保证精准度）

## 8. 数据库选择
- 中文库：CNKI、万方、维普
- 英文库：Web of Science、Scopus
覆盖该领域最核心、最权威的学术文献。
