# 检索式版本变更日志 query_changelog

## V0.1 (2026-03-26)
- 完成初始版本检索式设计，确立「方法 AND 任务 AND 场景 NOT 排除」的核心逻辑
- 构建方法词、任务词、场景词、排除词的基础术语对照表
- 确定时间范围：2015–2025
- 确定检索字段：标题、关键词、摘要
- 确定文献类型：期刊论文、硕博士论文、综述
- 本版本为项目基础框架，未适配具体数据库语法

## V0.2 (2026-03-27)
- 基于知网高级检索规则，将V0.1完整适配为可直接执行的知网检索语法
- 时间字段：使用 `YE BETWEEN ('2015','2025')` 精准限定发表年份
- 检索字段：统一使用知网 `TKA` 复合字段（标题+关键词+摘要）
- 方法词优化：补充 LSTM、长短期记忆网络、BiLSTM、attention LSTM 等中英文同义词
- 任务词优化：补充电力负荷预测、load forecasting、STLF 等中英文专业术语
- 场景词优化：补充电力系统、智能电网、配电网、power system 等场景词汇
- 排除词优化：新增交通流预测、服务器负载预测、bridge load prediction 等干扰项
- 本版本已在知网执行检索，获取有效文献 **46 条**，数据质量符合项目要求
