# Novelty Search v0

## 1. Literature Screening Comparison Table

本研究基于文献筛选结果，构建了一个系统性的对比表（literature screening comparison table），用于对已有研究进行结构化分析与横向比较。

该表格具有如下特点：

- 共包含 **25 个对比指标**
- 覆盖 **5 大分析维度**
- 用于支持后续的 **novelty identification（创新点识别）**

### 表格说明

该对比表主要从以下五个维度对文献进行分析：

| 维度 | 说明 |
|------|------|
| Research Objective | 研究目标与问题定义 |
| Methodology | 所采用的方法与模型 |
| Data & Scenario | 数据来源与应用场景 |
| Integration Strategy | 系统集成与优化策略 |
| Limitations | 现有研究的不足 |

该表通过对不同文献在上述维度上的表现进行对比，有助于识别研究空白与潜在创新方向。

## 3. 对比表的作用与意义

构建该对比表的主要目的在于：

### 3.1 系统化文献分析

通过统一的指标体系，将不同研究进行标准化描述，避免主观性分析，提高研究严谨性。

### 3.2 支持创新点挖掘

通过横向对比，可以识别：

- 方法上的不足（如缺乏动态优化）
- 数据层面的局限（如缺少多源数据）
- 系统集成的不足（如未考虑多能系统耦合）

从而为后续提出创新方法提供依据。

### 3.3 提高研究可复现性

CSV 格式的数据结构清晰，便于：

- 后续扩展文献
- 自动化分析
- 与其他工具（如Python、Gephi）结合使用
